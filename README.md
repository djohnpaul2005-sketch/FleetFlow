# 🚗 FleetFlow - Smart Fleet Management System

## 📌 Project Overview
FleetFlow is a web-based system designed to manage fleet operations efficiently. It helps administrators, managers, and drivers handle vehicles, trips, fuel usage, and reports in a centralized platform.

---

## 🚀 Features
- 👤 User Management (Admin, Manager, Driver)
- 🚗 Vehicle Management
- 🛣️ Trip Management
- ⛽ Fuel Tracking System
- 📊 Reports & Analytics
- 🔐 Secure Login System

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Other: Firebase (if used)

---

## 📂 Project Structure
- `/admin` → Admin panel
- `/driver` → Driver panel
- `/assets` → CSS, JS, images
- `/config` → Configuration files
- `/db` → Database scripts

---
